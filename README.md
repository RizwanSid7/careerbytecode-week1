## Bash Scripts

### 1. student_details.sh

This script accepts Name, College, and Branch as input and displays the details in formatted output.

Run command:

./scripts/student_details.sh

### 2. number_check.sh

This script checks whether a number is Positive, Negative, or Zero.

Run command:

./scripts/number_check.sh

### 3. numbers_1_to_20.sh

This script displays numbers from 1 to 20 using a for loop.

Run command:

./scripts/numbers_1_to_20.sh
